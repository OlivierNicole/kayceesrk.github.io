val transpose : 'a list list -> 'a list list
Transpose a matrix. For instance: 

      [ [ 1; 2];

        [ 3; 4]; ]

  

becomes: 

      [ [ 1; 3];

        [ 2; 4]; ]

  

val replace_one : ('a -> 'a option) -> 'a list -> 'a list option
replace_one f l replace the first element e in l such that f e is not None. The new value is y if f x = Some y.
