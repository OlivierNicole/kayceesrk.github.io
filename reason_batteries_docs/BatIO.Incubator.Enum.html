Print the contents of an enum, with first preceeding the first item (default: ""), last following the last item (default: "") and sep separating items (default: " "). A printing function must be provided to print the items in the enum. The flush parameter (default: false) should be set to true for the outer-most printing call. Setting inner calls to true - for example, for nested values - prevent indentation from working properly.

Example: pp ~flush:true Format.pp_print_int Format.std_formatter (1 -- 3)
